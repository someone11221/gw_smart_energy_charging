# GW Smart Charging

Integrace propojuje GoodWe se solárním forecastem a cenami elektřiny a optimalizuje nabíjení baterie.

Instalace
1. Add repository to HACS (type: Integration) — URL: https://github.com/someone11221/gw_smart_energy_charging
2. Install via HACS → Integrations.
3. Restart Home Assistant.
4. Add integration through Settings → Devices & Services → Add Integration → GW Smart Charging

Release notes and screenshots should be included in the GitHub Release body and in .github/assets for HACS UI to display.
