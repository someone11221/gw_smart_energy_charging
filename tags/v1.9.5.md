## Quick Summary

This release focuses on **maximizing cost savings** through intelligent timing and providing **complete visibility** into the system's 24-hour plan. The integration now analyzes electricity price trends and waits for the absolute cheapest time to charge, rather than starting at the first cheap opportunity.

## Installation

```bash
# Via HACS
1. Open HACS
2. Go to Integrations
3. Find "GW Smart Charging"
4. Click Update to v1.9.5

# Manual
1. Download release v1.9.5
2. Copy to custom_components/gw_smart_charging/
3. Restart Home Assistant
```
